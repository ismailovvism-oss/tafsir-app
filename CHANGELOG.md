## 2026-06 — Печатный мусхаф, настройки вида и навигация

### Добавлено
- **Печатный мусхаф Мадани (QPC v1).** Постраничные шрифты-глифы Комплекса
  им. короля Фахда: каждое слово — глиф, обведённый с печатной страницы.
  604 шрифта самохостятся в `fonts/qpc-v1/` и грузятся **лениво** (только
  открытая страница). Данные генерирует `build_qpc.py` → `data/qpc/v1/`.
- **Режим «Мусхаф» (постраничный).** Честная постраничная вёрстка: заголовки
  сур, басмала, выравнивание строк по ширине (построчный авто-подгон кегля,
  без «рек» из пробелов), концовки сур по центру. Переключатель **[Аяты | Мусхаф]**.
- **Шрифты арабского** (в ⚙ Вид): Насх, Каллиграфический (Усман Таха),
  Амири, **Печатный** (QPC), Куфи. Все самохостятся. По умолчанию — Печатный.
- **Панель ⚙ Вид** — единое место настроек отображения:
  - Оригинал: с огласовками / без огласовок / без точек (расм).
  - Размер шрифта — **раздельно** арабский и русский.
  - Межстрочный интервал — **раздельно** арабский и русский.
  - Поля (отступы по краям).
  - Размер боковой панели (суры, теги).
  - Кнопка «↺ По умолчанию».
- **Навигация (нижняя панель)** в обоих режимах: единица
  (Сура / Страница / Джуз / Хизб) + список выбора + стрелки «‹ ›».
  Свайп влево/вправо листает. В режиме «Мусхаф» стрелки всегда листают
  страницы. Заголовок показывает «Сура · Джуз N · стр. P/604».
- **Markdown в тафсирах/переводах**: `**жирный**`, `*курсив*`, `` `моно` ``,
  `> цитата`, `# заголовки`, списки, **таблицы** `| |`, линия `---`.
- **Сворачивание длинных тафсиров** (>600 символов) с кнопкой «Развернуть».

### Исправлено
- Арабский не отображался на GitHub Pages (Jekyll вырезал файлы на `_`) —
  добавлен `.nojekyll`.
- В узкой/портретной ориентации строки мусхафа вылезали за край — авто-подгон
  кегля теперь и уменьшает строки; пересчёт при смене ширины (ResizeObserver).
- Стрелки навигации листали только суры — позиция теперь отслеживается в обоих
  режимах (`ST.mushafPage`).

## Ранее — База (Phase 1–6)

- **Phase 1.** Разрезка данных на чанки по сурам `data/tafsirs/<id>/<surah>.json`,
  ленивая загрузка `loadChunk`, LRU-кэш.
- **Phase 2.** Мобильные улучшения: видимые контролы без hover, пинч-зум,
  регулятор размера, мягкое анти-копирование.
- **Phase 3.** Виртуализация чтения (гидрация аятов по IntersectionObserver),
  дешёвое переключение тафсиров.
- **Phase 4.** URL-хеш-роутинг `#сура[:аят]`, deep-links.
- **Phase 5.** Предсобранный инвертированный индекс поиска.
- **Phase 6.** Правки поддерживаемости.
- Несколько вариантов арабского текста (огласовки / без огласовок / расм),
  опциональный куфи (Reem Kufi), редизайн правой панели (фасеты + флайаут).

См. `BACKLOG.md` для детального чек-листа фаз.
