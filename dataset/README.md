# Пословный русский глосс Корана (Quran Word-by-Word Russian Gloss)

Контекстный **пословный перевод на русский язык** каждого слова Корана —
по одному русскому глоссу под каждым арабским словом, в правильных падежах и
окончаниях с учётом контекста аята.

- **Полнота:** весь мусхаф — 114 сур, 6 236 аятов, **77 429 токенов** (100%).
- Каждый токен переводился **индивидуально, по контексту** (одно и то же
  арабское слово передаётся по-разному в зависимости от смысла — напр. مِن как
  «из», «от», «среди»).
- Стиль — буквальный пословный, со **сверкой по переводу И. Ю. Крачковского**
  (где Крачковский шёл окольно, бралось прямое значение, если смысл не ломается).

### Формат JSON

```json
{
  "1:1:1": "Во имя",
  "1:1:2": "Аллаха",
  "1:1:3": "Милостивого",
  "1:1:4": "Милосердного",
  "112:1:1": "скажи"
}
```

Ключ — `surah:ayah:word` (номера от 1). Нумерация слов соответствует
стандартной пословной разбивке Корана (Tanzil / Quranic Arabic Corpus).

### Формат TSV

```
surah	ayah	word	ru
1	1	1	Во имя
1	1	2	Аллаха
```

Знак `—` (тире) означает токен без отдельного русского эквивалента
(служебные связки идафы, дубли при разбиении, английские структурные слова).
