# Tafsir Library — Backlog

Цель: удобная платформа для чтения тафсиров. Сотни тафсиров/переводов.
Особый акцент на мобильной версии: удобно, красиво, плавно. Лёгкое переключение
тафсиров и чтение со смартфона.

Реализуется по порядку. Каждый пункт — функциональный и тестируемый, приложение
остаётся рабочим после каждого шага.

---

## Phase 5 — Поиск на масштабе ✅
- [x] 5.1 `build_index.py` строит инвертированный индекс `data/index/<id>.json`
      (слово → "сура:аят ..."). Поиск грузит индекс (не полный текст), находит аяты
      по словам запроса (AND + подстрока по словам), текст для показа берёт из чанков
      только для найденных аятов. Fallback на полный скан, если индекса нет.
      Проверено: «милостивого» → 27 результатов, грузится index/kuliev.json, не монолит.
- [x] Побочно исправлен баг `normalizeArabic`: диапазон `ؐ-ٟ` захватывал
      буквы (0x621–0x64A) и стирал весь арабский текст → арабский поиск был сломан.
      Теперь корректные диапазоны огласовок (в index.html и build_index.py).
- Примечание: «искать во всех» при сотнях тафсиров грузит все индексы — на больших
  объёмах в итоге нужен бэкенд/серверный индекс.
