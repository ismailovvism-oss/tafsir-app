# Tafsir Library — Backlog

Цель: удобная платформа для чтения тафсиров. Сотни тафсиров/переводов.
Особый акцент на мобильной версии: удобно, красиво, плавно. Лёгкое переключение
тафсиров и чтение со смартфона.

Реализуется по порядку. Каждый пункт — функциональный и тестируемый, приложение
остаётся рабочим после каждого шага.

## Phase 5 — Поиск на масштабе ✅
- [x] 5.1 `build_index.py` строит инвертированный индекс `data/index/<id>.json`
      (слово → "сура:аят ..."). Поиск грузит индекс (не полный текст), находит аяты
      по словам запроса (AND + подстрока по словам), текст для показа берёт из чанков
      только для найденных аятов. Fallback на полный скан, если индекса нет.
      Проверено: «милостивого» → 27 результатов, грузится index/kuliev.json, не монолит.
- [x] Побочно исправлен баг `normalizeArabic`: диапазон `ؐ-ٟ` захватывал
      буквы (0x621–0x64A) и стирал весь арабский текст → арабский поиск был сломан.
      Теперь корректные диапазоны огласовок (в index.html и build_index.py).
- Примечание: «искать во всех» при сотнях тафсиров грузит все индексы — на больших
  объёмах в итоге нужен бэкенд/серверный индекс.

## Phase 7 — Тафсир Ибн Касир (рус.) — доработки
Импорт сделан (`build_kathir_ru.py`, суры 1–19 и 59–114) — см. CHANGELOG 2026-06.
Осталось:
- [ ] **Недостающие суры 9–10 и 20–58.** Нужны TXT недостающих томов Мухтасара;
      лучшая версия экстракции — серия файлов `80179…` (формат B: двухколоночный,
      номера аятов, чистый арабский с огласовками). Серия `8865…/8933…/9497…` —
      старая (формат A, битый арабский).
- [ ] **Инлайн-цитаты арабского в комментарии** сейчас вырезаны (держимся на
      русском глоссе «…»). Можно ре-сорсить из `_arabic.json` сопоставлением
      фрагментов — нетривиально.
- [ ] **Сноски формата A**: маркеры распознаются эвристикой (цифра, приклеенная
      к слову) — возможны редкие ложные срабатывания, стоит выборочно проверить.
- [x] **(app) «У аята — все доступные тафсиры»** — реализовано **Системой покрытия**
      (а не отдельной фичей): бейдж `🌐 N · 📚 M` в шапке аята → попап со списком
      источников у аята (✓/—, группы Переводы/Тафсиры), действия «только здесь»
      (разовый инлайн-показ) и «постоянно» (включить всюду). Шире исходной задумки
      (включены и переводы, не только `kind:"tafsir"`).
- [x] **(app) Главная страница** — пятый режим (🏠) наряду с Аяты/Слова/Мусхаф/Источник;
      экран по умолчанию при открытии без deep-link (хеш `#home`). Блок «Продолжить
      чтение» (из `lastPos`) + «Недавнее» (лента `recent`, MRU, до 5 сур, по одной на
      суру) + настраиваемые быстрые ссылки. Синтаксис: `N` (вся сура) / `N/A` (аят) /
      `N/A-B` (диапазон), через запятую/строку, + опц. своя подпись; подсказка в поле.
      Дефолтный список из распространённых сур. **Пятница** (`getDay()===5`) →
      Аль-Кахф выделенной плиткой первым. Хранилище: `tl_recent`, `tl_quickLinks`.
- [x] **(app) Экспорт/импорт личных данных** — один `.json` (заметки + закладки +
      теги + дерево тегов) для бэкапа/переноса. Кнопки в панели 📚 под «Мои заметки».
      Импорт = слияние: новые добавляются, теги объединяются, дерево мёрджится по
      именам; при конфликте заметок — диалог (Оставить мои / Дописать обе / Заменить).
